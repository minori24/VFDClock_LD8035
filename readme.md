# LD8035E VFD時計キット by Fablab Kannai

Fablab関内で開発した、VFD時計のキットです。Arduino Nanoをベースに、カシオミニで使われていたVFD管LD8035Eを6管点灯させるレトロな時計です。パーツ数を抑えたシンプルな構成ですが、ミニゲームやタイマーもついた高機能な時計です。
